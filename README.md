## Technologies Implemented 

* Javascript
* React
* Redux
* Auth0
* Material UI
* Express 
* Sequelize
* Postgress SQL

## Installation

npm install 

## Usage

npm run dev - or better than this you can go to https://app-gym-frontend.vercel.app
